To start game shoot in any of 3 directions with LMB;
Left and right arrow keys to move platforml;
Buff creates wall, which prevents ball from falling;
Debuff kills player;
Command line start example:
Arcanoid.exe 1200 800
To win destroy all blocks and catch the ball.
